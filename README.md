# Server
Official source code of the Meower server, written in Python. Powered by CloudLink. 

## Running
```py
git clone https://github.com/meower-media/server.git --recursive
cd Meower-Server
cd Meower-Server
pip install -r requirements.txt

cp .env.example .env

# edit env files

python3 main.py
```
